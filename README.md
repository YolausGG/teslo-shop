# Teslo API

1. Clone project
2. ```npm install```
3. Clone the ```.env.template``` file and rename it to ```.env```
4. Change the environment variables
5. Build the database
```
docker-compose up -d
```
---
6. Run ```npm run start:dev```
---
7. Execute SEED
```
http://localhost:3001/api/seed
```
